# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-27

First tagged release of the AI-Assisted Development Starter Kit. Consolidates the bootstrap, Jira integration, cleanup, and knowledge-layer restructuring completed since project inception.

### Added
- Initial starter kit with AI knowledge layers under `.agents/` (`sources/`, `memory/`, `reference/`, `specs/`, `plans/`)
- Jira skill with mcp-atlassian Cloud integration — create / edit / search / transition / comment / link Epics, Tasks, Bugs (bulk supported)
- Knowledge-layer restructuring for slim `CLAUDE.md` and on-demand context:
  - `.agents/memory/architecture.md` — directory map regenerated by `/create-CLAUDE_MD`
  - `.agents/memory/project-brief.md` — distilled PRD TL;DR regenerated by `/refresh-brief`
  - `.agents/memory/domain/business-model.md` — pricing / billing facts seeded by `/refresh-brief`
  - `/refresh-brief` command (PRD → brief and optional business-model seed)
  - `/prime` quick (default) and `/prime full` modes with status-aware loading
  - `/prime-ba` for Business Analyst priming (PRD, specs, Jira backlog)
  - Status frontmatter convention (`empty | seeded | populated`) — placeholders are skipped on load
  - `When to Read` routing table in `.agents/memory/index.md`
  - `CLAUDE.md` hard cap at 200 lines; routing pushed to memory layer
- Security-first `.claude/settings.json` — non-destructive git allowed, destructive ops denied, secret / credential writes blocked, audit-log hooks
- `documentation-manager` sub-agent

### Maintenance
- Bootstrap artifact cleanup
- Removed personal data and hardcoded absolute paths from the shipped starter
